# Assets Directory

This directory contains icon files for the ExcelBot Pro add-in.

## Required Icons

The following icon files are required for the Office Add-in:

- **icon-16.png** - 16x16 pixel icon for ribbon
- **icon-32.png** - 32x32 pixel icon for ribbon
- **icon-64.png** - 64x64 pixel icon for high-resolution displays
- **icon-80.png** - 80x80 pixel icon for ribbon

## Icon Requirements

- Format: PNG with transparent background
- Design: Should represent the ExcelBot Pro brand
- Colors: Consistent with the purple gradient theme (#667eea to #764ba2)
- Content: Can include a robot or automation symbol

## Creating Icons

You can create these icons using:
- Adobe Illustrator or Photoshop
- GIMP (free alternative)
- Online icon generators
- Canva

For now, placeholder icons are provided. Replace them with proper branded icons before distribution.
